v2.0.0 — Courante ✓ Stable IA Médicale
v2.0.0 Major ● Courante 2025-03-15
Ajouté
  • Moteur de classification hybride — Choix entre mots-clés locaux et API Claude (claude-sonnet-4-20250514) pour une classification contextuelle avancée.
  • Intégration Telegram native — Alertes automatiques après chaque traitement. Bouton de test intégré dans l'interface.
  • Journal de session persistant — Enregistrement horodaté de chaque traitement avec catégorie, durée et identifiant de session.
  • Pipeline de workflow visuel — Barre d'état temps réel sur 8 étapes : START → INIT → LECTURE → CLASSIFICATION → ACTION → JOURNALISATION → CLÔTURE → END.
  • Compteurs par catégorie — Statistiques en temps réel des 4 types de traitement.
  • Champ pièces jointes — Noms de pièces jointes intégrés dans le contexte de classification.
  • Copie rapide — Bouton ⎘ pour copier instantanément la réponse générée dans le presse-papier.
  • Panneau de statut système — État du moteur, de l'API Claude et de Telegram affiché en temps réel.
Modifié
  • Interface entièrement redessinée — esthétique bleu professionnel, typographie claire et animations fluides.
  • Génération de réponse améliorée — intégration de la signature personnalisée dans chaque brouillon généré.
  • Logique de classification renforcée — meilleure discrimination entre INFO_REQUIREMENT et PROOF_OF_STATUS.
  • Architecture du workflow redessinée — transitions d'état robustes avec gestion des erreurs à chaque étape.
AVIS
Version majeure : rupture de compatibilité avec les configurations v1.x. Les paramètres de session antérieurs ne sont pas migrés automatiquement.
v1.4.2 Patch 2025-01-30
Corrigé
  • Correctif du bug d'horodatage EST/EDT — les entrées du journal affichaient l'heure UTC incorrectement.
  • Correction de la condition de course lors de traitements rapides successifs dans le journal.
  • L'état CLÔTURE ne se déclenchait pas systématiquement en cas d'erreur réseau Telegram — résolu.
Modifié
  • Format du journal enrichi : ajout du numéro de session et de la durée de traitement en millisecondes.
