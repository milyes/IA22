UCI

Universal Chat UI

Interface universelle pilotée par IA — centralisez Slack, Discord, GitHub, Jira et workflows IA.

MIT v1.0.0 build: passing

Un hub — toutes vos conversations

Universal Chat UI unifie messages, tickets et outils de développement dans une seule interface contextuelle. Gagnez du temps grâce aux automatismes IA, plugins et intégrations.

Intégrations
Slack, Discord, Telegram, GitHub, Jira, et plus.
Assistant IA
Réponses contextuelles, suggestions proactives et actions automatisées.
Plugins modulaires
Ajoutez des connecteurs et commandes personnalisées en quelques lignes.
Sécurité
OAuth, permissions granulaires et chiffrement de bout en bout.

Installation

Démarrage local (Node.js) ou conteneurisé (Docker).

Node.js

git clone https://github.com/yourusername/universal-chat-ui.git
cd universal-chat-ui
npm install
npm run dev

Docker

docker-compose up -d --build

Variables d'environnement

# .env
SLACK_TOKEN=your_slack_token
DISCORD_TOKEN=your_discord_token
TELEGRAM_TOKEN=your_telegram_token
GITHUB_TOKEN=your_github_token
JIRA_API_KEY=your_jira_key

Créer un plugin

Déposez vos modules dans `plugins/` — exemple minimal :

module.exports = {
  name: "hello-world",
  trigger: "!hello",
  execute: async (ctx) => {
    await ctx.reply("Hello from my custom plugin!");
  }
};

Redémarrez le service après ajout.
