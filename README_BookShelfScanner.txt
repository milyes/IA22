
# 📚 BookShelfScanner

**Auteur :** Zoubirou Mohammed Ilyes  
**ORCID :** https://orcid.org/0009-0007-7571-3178  
**Version :** 1.0.0  
**Plateforme :** NetSecurePro IA

## 🔍 Description
BookShelfScanner est une application intelligente permettant :
- La **détection de livres** dans des fichiers PDF.
- L’**analyse automatique** de contenu.
- La **recommandation IA** basée sur les préférences de lecture.
- L’affichage local via une interface Web intégrée.

## 📦 Contenu du Package
- `BookShelfScanner.apk` — Application mobile Android Flutter
- `index_bookshelfscanner.html` — Interface WebView intégrée
- `BookShelfScanner_Official_Fiche.pdf` — Fiche PDF officielle du projet
- `bookshelfscanner_qr.png` — QR code de lancement
- `README.txt` — Ce fichier

## 🚀 Instructions d'installation
1. Copier l’APK sur votre mobile Android.
2. Activer l’installation de sources inconnues.
3. Installer puis lancer l’application.
4. L’interface IA apparaîtra avec les boutons : Scanner, Analyser, Recommander.
